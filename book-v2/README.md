# 논증의 정원 v2

부제: 우리 사회가 학습시킨 글쓰기 습관으로부터 논증적 글쓰기를 다시 세우는 단행본 재구성안

이 버전은 기존 단행본 초고를 단순 보완한 것이 아니라, 책의 진입점과 장 배치를 다시 설계한 재구성본이다. 핵심 아이디어는 분명하다. 학생들은 대개 “논증이 무엇인지 몰라서”만 실패하지 않는다. 오히려 입시, 면접, 토론대회, 시중 논술 강의, 온라인 글쓰기 조언, 그리고 때로는 학계의 관행이 보상해 온 습관을 성실하게 따르다가 실패한다. 이 버전은 먼저 그 사회적 배경을 보여 주고, 그 배경이 학생 글에 어떤 무의식적 습성을 남겼는지 설명한 뒤, 각 장이 그 구조적 실패를 하나씩 해부하고 교정하도록 재조직한다.

## 재구성 원리

- 각 장은 원리 설명보다 먼저 그 실패를 낳은 사회적 배경과 글쓰기 습관을 제시한다.
- 전체 흐름은 `배경 진단 → 내면화된 습성 파악 → 구조적 실패 인식 → 논제 엄정화 → 독해와 쟁점화 → 본론 재구성 → 서론·결론 교정 → 반론과 추론 유형 → 장문 설계 → 워크북`의 구조를 따른다.
- 본론 장은 “왜 학생 글이 그럴듯하게 실패하는가”를 책의 중심 문제로 삼는다.
- 서론·결론, 반론, OREO는 모두 “실패를 줄이는 기능적 장치”로 재배치한다.
- 각 장은 향후 강연, 워크숍, 온라인 미니 강의와 연결될 수 있도록 `실패 장면`, `교정 원리`, `실전 과제`의 세 층을 가진다.
- 현재 기준 원고는 Jekyll 호환을 위해 [`_includes/book-src-v2/`](/Users/jeyounson/2026SPRING/_includes/book-src-v2)에 두고 관리한다.
- `book-v2/` 아래의 개별 장 파일은 기준 원고를 불러오는 래퍼(wrapper) 파일이다.
