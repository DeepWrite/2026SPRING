# 섹션별 분석

아래에서는 **Garrett Hardin (1968) 「The Tragedy of the Commons」**의 구조를 **섹션별로 핵심 Thesis(명제)**와 **이를 지지하는 Argument(논변)**의 형태로 분석한다.
가능한 한 **논증적 글쓰기 분석 형식(Thesis → Argument 구조)**으로 정리하였다. 

---

## 1. Introduction: **No Technical Solution Problems**

### Thesis

어떤 사회문제들은 **기술적 해결책(technical solution)**으로 해결될 수 없으며, **인간의 가치·규범·제도의 변화가 필요하다.**
그리고 **인구 문제(population problem)**는 이러한 **“no technical solution problem”**에 속한다.

### Arguments

#### Argument 1: 개념 규정

* **Technical solution** = 자연과학적 기술 변화만 요구하며 인간 가치나 도덕을 바꾸지 않는 해결책.
* 그러나 일부 문제는 이런 방식으로 해결될 수 없다.

#### Argument 2: 사례 (tic-tac-toe)

* 완벽한 상대와의 틱택토에서 **“이기는 방법”**은 없다.
* 이기려면

  * 게임 규칙을 깨거나
  * 게임 자체를 포기해야 한다.
* 즉 **문제의 구조 자체를 바꾸지 않으면 해결 불가능**.

#### Argument 3: 인구 문제에 대한 적용

* 많은 사람들이

  * 해양농업
  * 새로운 작물
  * 기술혁신
    등으로 인구 문제를 해결할 수 있다고 믿는다.
* 그러나 이는 **문제의 구조를 오해한 것**이다.

#### Conclusion

따라서 **인구 문제는 기술 문제가 아니라 사회적·도덕적 제도 문제이다.**

---

## 2. **What Shall We Maximize?**

### Thesis

**인구를 최대화(maximize population)** 하는 목표는 **사회적 복지를 최대화할 수 없다.**
따라서 **최적 인구(optimum population)**는 **최대 인구보다 작다.**

### Arguments

#### Argument 1: 유한 세계 논증

1. 인구는 **기하급수적으로 증가**한다 (Malthus).
2. 그러나 세계는 **유한하다**.
3. 따라서 **1인당 자원은 감소**한다.

→ 무한한 인구 증가는 불가능.

---

#### Argument 2: 다중 변수 최대화 불가능

Bentham의 명제

> “the greatest good for the greatest number”

는 논리적으로 문제 있다.

이유:

* **두 변수(인구, 행복)**를 동시에 최대화할 수 없다.
* 수학적으로 다변수 최대화는 불가능.

---

#### Argument 3: 생물학적 에너지 논증

인간의 에너지 소비

* maintenance calories
* work calories

인구를 최대화하려면

```
work calories → 0
```

즉

* 문화
* 예술
* 여가
* 지적 활동

모두 제거해야 한다.

→ **인구 최대화는 인간적 삶을 파괴한다.**

---

#### Argument 4: 가치 비교 문제

“좋은 것”은 사람마다 다르다.

예:

* wilderness vs ski resorts
* estuary vs factory

하지만 현실에서는 **가치 선택이 이루어진다.**

→ 인간 사회는 **명시적 가치 weighting 체계**를 만들어야 한다.

---

#### Conclusion

문제는

> **최대 인구가 아니라 최적 인구를 찾는 것**

이다.

---

## 3. **Tragedy of Freedom in a Commons**

### Thesis

**공유 자원(common resource)**에서 **개인의 합리적 선택은 집단적 파멸을 초래한다.**

### Arguments

#### Argument 1: Lloyd의 목초지 모델

상황:

* 목초지 commons
* 여러 목동

각 목동의 계산:

```
+1 : 추가 소의 이익 (전부 개인 소유)
-1/n : 과방목 비용 (모두에게 분산)
```

결론:

```
+1  >  -1/n
```

→ 합리적 행동 = 소를 늘린다.

모든 목동이 같은 행동

→ 과방목
→ 목초지 파괴.

---

#### Argument 2: 구조적 딜레마

각 개인:

> 합리적으로 행동

전체 결과:

> 자원 붕괴

즉

**개인 합리성 → 집단 비합리성**

---

#### Argument 3: 실제 사례

Hardin은 여러 사례 제시:

* 미국 서부 목축
* 바다 어업
* 국립공원
* 해양 포경

모두 동일 구조:

```
공유자원 + 개인이익 추구 → 자원 고갈
```

---

#### Conclusion

> **Freedom in a commons brings ruin to all.**

---

## 4. **Pollution**

### Thesis

공유지 비극은 **자원 채취뿐 아니라 오염 문제에서도 발생한다.**

### Arguments

#### Argument 1: 구조적 대칭

Commons 구조

| 유형                | 행동  |
| ----------------- | --- |
| resource commons  | 가져감 |
| pollution commons | 버림  |

개인의 계산:

```
pollution cost_shared < purification_cost
```

따라서

→ 오염을 선택.

---

#### Argument 2: 인구 증가와 오염

인구가 적을 때

* 자연이 흡수

인구가 많을 때

* 자연 정화 능력 초과

→ 제도 변화 필요.

---

#### Argument 3: 해결책

공유지 해결 방식

1. private property
2. coercive law
3. taxation

---

#### Conclusion

공유지 문제는 **제도적 개입 없이는 해결 불가능**.

---

## 5. **How to Legislate Temperance?**

### Thesis

도덕 규칙은 **행위 자체가 아니라 시스템 상태(system state)에 의존한다.**

### Arguments

#### Argument 1: 상황 윤리

행위의 도덕성

```
morality = function(system state)
```

예:

* frontier waste → harmless
* metropolis waste → disastrous

---

#### Argument 2: 법 제도의 한계

전통 윤리

```
Thou shalt not
```

그러나 현대 사회는

* 복잡
* 인구밀집
* 기술발달

→ 세부 규칙 필요.

---

#### Argument 3: 행정법 필요성

현대 해결 방식

```
statutory law + administrative law
```

하지만 위험:

> Who watches the watchers?

---

## 6. **Freedom to Breed Is Intolerable**

### Thesis

**번식의 자유(freedom to breed)**는 **공유지 비극의 핵심 원인이다.**

### Arguments

#### Argument 1: 자연 상태

자연에서는

```
overbreeding → offspring death
```

→ 자연적 피드백 존재.

---

#### Argument 2: 복지국가 문제

현대 사회

* welfare state
* 생존 보장

→ 자연적 피드백 제거.

따라서

```
overbreeding → population explosion
```

---

#### Argument 3: 인권 선언 비판

UN 선언

> 가족 규모 결정은 가족의 권리

Hardin의 주장

→ 이는 공유지 비극 구조를 강화.

---

## 7. **Conscience Is Self-Eliminating**

### Thesis

**양심에 호소하는 방식은 장기적으로 실패한다.**

### Arguments

#### Argument 1: 선택 효과 (Darwinian selection)

양심적 사람

→ 출산 제한

비양심적 사람

→ 출산 증가

따라서

```
generation → conscience 감소
```

---

#### Argument 2: 진화 논증

결과

```
Homo contracipiens → extinct
Homo progenitivus → dominant
```

---

## 8. **Pathogenic Effects of Conscience**

### Thesis

양심 호소는 **심리적·사회적 부작용을 낳는다.**

### Arguments

#### Argument 1: double bind

개인이 받는 메시지

1. 제한하라 (공적 요구)
2. 남들은 exploit (암묵적 메시지)

→ Bateson의 **double bind**

---

#### Argument 2: 죄책감의 문제

Nietzsche

> bad conscience = illness

Goodman

> guilt produces anxiety

---

## 9. **Mutual Coercion Mutually Agreed Upon**

### Thesis

공유지 비극의 해결책은

> **mutual coercion mutually agreed upon**

즉 **집단적으로 합의된 강제 제도**이다.

---

### Arguments

#### Argument 1: 은행 강도 비유

은행을 commons로 취급하면

→ robbery

해결:

```
law + punishment
```

---

#### Argument 2: 세금과 규제

예:

* parking meters
* taxation

→ **선택 비용을 조정**

---

#### Argument 3: 자유와 강제

Hardin 주장:

```
coercion ≠ tyranny
coercion = collectively agreed rules
```

---

## 10. **Recognition of Necessity**

### Thesis

인류는 점점 더 **commons를 포기하는 방향으로 진화해 왔다.**

그리고

> **출산의 자유도 포기해야 한다.**

---

### Arguments

#### Argument 1: 역사적 enclosure

commons → 폐지 사례

* 농지
* 사냥
* 어업
* 폐기물 처리

---

#### Argument 2: 자유의 재정의

Hegel 인용:

> Freedom is the recognition of necessity.

---

#### Argument 3: 최종 결론

```
Freedom to breed → ruin
Mutual coercion → survival
```

---

## 전체 논문의 핵심 논증 구조

Hardin의 전체 논증은 다음 **5단계 구조**로 요약된다.

```
1 기술 해결책 불가능
2 인구 최대화는 비합리적
3 공유지 구조는 파멸을 낳음
4 양심 호소는 실패
5 해결책 = 합의된 강제 제도
```

즉

```
Commons + Individual Rationality
        ↓
Collective Ruin
        ↓
Solution = Institutional Constraint
```

---

## 철학적 구조 (한 문장 정리)

Hardin의 핵심 테제:

> **개인의 합리성이 공유 자원 조건에서 집단적 파멸을 낳기 때문에, 자유를 제한하는 집단적 제도가 필요하다.**
